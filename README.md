# Portofolio Backend

Backend aplikasi portofolio pribadi yang dibangun menggunakan **Laravel** sebagai bagian dari proses belajar **Software Engineering**.

Project ini dikembangkan **bertahap**, dimulai dari CRUD sederhana hingga REST API dan authentication.

---

## 🎯 Purpose

Project ini dibuat sebagai:
- Media belajar Software Engineering
- Implementasi best practice Laravel
- Backend untuk portofolio frontend
